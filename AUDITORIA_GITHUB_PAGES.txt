Como publicar no GitHub Pages:
1. Crie um repositório no GitHub.
2. Envie todo o conteúdo deste ZIP para a raiz do repositório, mantendo a pasta docs/.
3. Em Settings > Pages, selecione Deploy from branch, branch main, pasta /root.
4. Abra a URL gerada pelo GitHub Pages. Use index.html como entrada.

Observação: em GitHub Pages, os PDFs abrem por HTTPS, então funciona melhor em celular/tablet do que file:// ou content://.
